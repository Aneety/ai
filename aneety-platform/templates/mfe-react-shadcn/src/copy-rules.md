# Regras de copy para microfrontends Aneety

Use linguagem de produto e tarefa.

## Preferir

- Configuração precisa de atenção.
- Ação pendente de envio.
- Recomendação operacional.
- Comprovante disponível.
- Custódia protegida.
- Permissão insuficiente.
- Acesso expirado.

## Evitar em UI final

- API, chave, token, hash, adapter, banco, worker, runtime, framework, provider, algoritmo, stack, segredo.
- IDs técnicos de operadores ou ferramentas internas.
- Promessa automática não validada por contrato.
